请将登录所需的模板图片放在此目录中，并保持文件名与 config.toml 中一致。

推荐流程：
1. 使用截图工具截取按钮或输入框的“紧凑区域”，尽量只包含控件本体。
2. 保存为 PNG 格式，分辨率与实际屏幕缩放保持一致。
3. 如果控件有“激活态/选中态”，建议同时提供普通态与激活态。
4. 如果左右两边都可能出现“上课”按钮，可分别截图为 on_course.png 与 on_course_right.png。

默认文件名（单模板场景）：
- sidebar_button.png
- on_course.png
- account_input.png
- password_input.png
- login_button.png
